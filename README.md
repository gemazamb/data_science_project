# README - chexpert_preview.ipynb
## Descripción
Este notebook contiene el análisis exploratorio y visualización de datos del conjunto CheXpert, enfocado en datos demográficos y etiquetas de enfermedades radiológicas. Incluye carga de datos, limpieza, análisis de valores nulos y gráficos de proporciones de etiquetas.

## Requisitos
- Python 3.8+
- pandas
- matplotlib
- seaborn
- openpyxl

## Uso
1. Descarga el notebook `proyect(i).ipynb`.
2. Ejecuta cada celda en orden.
3. Modifica las rutas de los archivos si es necesario.

## Estructura
- Carga de datos demográficos y etiquetas.
- Limpieza y análisis de valores nulos.
- Visualización de proporciones de etiquetas.
- Gráficos personalizados.
